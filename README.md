# 마켓 대시보드

미국 및 한국 주식시장을 위한 AI 기반 종합 투자 분석 플랫폼입니다.

## 기술 스택

**백엔드**
- Node.js + Express
- node-cron (스케줄러)
- node-cache (인메모리 캐시)
- Axios (HTTP 클라이언트)
- Cheerio (웹 스크래핑)

**프론트엔드**
- React 18 + Hooks
- Tailwind CSS
- Recharts (차트)
- TradingView Lightweight Charts (고급 캔들 차트)

## 설치 및 실행

### 1. 패키지 설치

```bash
cd backend && npm install
cd ../frontend && npm install
```

### 2. 환경 변수 설정

```bash
cp backend/.env.example backend/.env
```

`backend/.env` 파일을 열어 API 키를 입력합니다.

```env
FINNHUB_API_KEY=여기에_입력
ANTHROPIC_API_KEY=여기에_입력
FRED_API_KEY=여기에_입력
ALPHA_VANTAGE_API_KEY=여기에_입력
DART_API_KEY=여기에_입력
KIS_APP_KEY=여기에_입력
KIS_APP_SECRET=여기에_입력
KIS_ACCOUNT_NO=계좌번호
PORT=3001
```

### 3. 개발 모드 실행

터미널 두 개를 열어 각각 실행합니다.

```bash
# 터미널 1 - 백엔드 (포트 3001)
cd backend
npm run dev

# 터미널 2 - 프론트엔드 (포트 8080)
cd frontend
npm run dev
```

브라우저에서 `http://localhost:8080` 접속

### 4. 프로덕션 빌드

```bash
cd frontend
npm run build
```

## 화면 구성

### 미국 시장 탭

**상단 - 시장 지수 바**
- S&P 500, Nasdaq 100, Dow Jones, Russell 2000, VIX
- 5분 간격 자동 갱신

**1행 - 매크로 / 뉴스 / 경제 캘린더**

| 패널 | 내용 |
|------|------|
| 매크로 개요 | 금, WTI, 브렌트, 구리, 은 / VIX, 달러인덱스 / 2년/20년 국채 수익률 |
| 뉴스 피드 | Finnhub 실시간 헤드라인 + 긍정/부정 감성 분석 |
| 경제 캘린더 | 향후 14일 미국 주요 이벤트 + 실적 발표 20개 |

**2행 - 섹터 히트맵 / AI 모닝 브리핑**

| 패널 | 내용 |
|------|------|
| 섹터 히트맵 | GICS 11개 섹터 ETF 일간 수익률 색상 표시 |
| AI 모닝 브리핑 | Claude가 지수/섹터/뉴스/경제지표를 종합한 한국어 시장 분석 (매일 자동 생성) |

**3행 - 매크로 대시보드**

10개 카테고리 탭 (인플레이션 / 통화 / 금리 / 고용 / 경기 / 소비 / 주택 / 원자재 / 글로벌 / 시장심리)

각 탭마다 4~6개 FRED 경제지표 카드가 표시됩니다.
- 초록: 강세 조건 충족
- 빨강: 약세 조건 충족
- 노랑: 중립
- 지표 클릭 시 5년 히스토리 차트 + Claude 해석 모달

**4행 - 미국 주식 스크리너**

S&P 100 + 주요 종목 대상, RSI 및 60일 등락폭 기준 필터링

---

### 한국 시장 탭

**1행 - 지수 / 섹터 / 뉴스 / 캘린더**

| 패널 | 내용 |
|------|------|
| 한국 지수 | KOSPI, KOSDAQ, KOSPI 200 |
| 섹터 ETF | 반도체, IT, 금융, 헬스케어 등 10개 |
| 뉴스 | 한국 시장 뉴스 20건 + 감성 분석 |
| 캘린더 | BOK 금통위 일정 + 실적 발표 |

**2행 - 매크로 대시보드**

미국 탭과 동일한 FRED 경제지표 + Caixin 중국 PMI 포함

**3행 - MVP 테이블**

아이투자 사이트에서 1년 이내 MVP 선정 기사를 자동 수집하여 분기별 언급 빈도를 집계합니다.

**4행 - PEG 퀀트 스크리너**

Z-Score 기반 복합 팩터 점수로 섹터 내 종목 순위를 산출합니다.

**5행 - 한국 종목 스크리너**

KIS API 기반 RSI + 등락폭 스크리닝 결과 (매일 00:30 자동 갱신)

---

### 기업가치평가 탭

3단계 마법사 형태로 진행됩니다.

1. 종목 선택 및 재무 데이터 자동 로드 (DART 기반)
2. WACC / 성장률 파라미터 설정
3. 결과 확인 (DCF + 상대가치 + 민감도 분석) 및 히스토리 저장

---

### 종목 점수 탭

6자리 종목코드 또는 자동완성 검색으로 종목을 선택하면 44개 항목을 분석한 종합 대시보드가 표시됩니다.

## 스크리너 상세

### 미국 주식 스크리너 `/api/screener`

**대상**: S&P 100 + 주요 ETF 약 80종목

**계산 지표**

| 지표 | 설명 |
|------|------|
| RSI(14) | Wilder 방식 14일 RSI |
| 60일 등락폭 | (60일 고점 - 저점) / 저점 x 100 |
| ATR(14) | 평균 진폭 |
| 거래량 비율 | 당일 거래량 / 20일 평균 |
| 가격 위치 | 60일 범위 내 현재가 위치 (%) |

**기본 필터 파라미터** (쿼리 파라미터로 조정 가능)

```
rsiMin=20   RSI 하한
rsiMax=60   RSI 상한
rangeMin=50 60일 등락폭 최소값 (%)
```

### KIS 5가지 기법 스크리너 `/api/kis-screener`

매일 01:00 전종목 지표를 자동 계산하고 디스크에 저장합니다.
프론트엔드에서 저장된 지표값을 받아 클라이언트 측에서 실시간 조건 필터링을 수행합니다.

**지원 기법**

| 기법 | 핵심 지표 | 개념 |
|------|---------|------|
| 밥그릇 | MA 이격도, 구름대/CCI 돌파, 볼린저 밴드 | 장기 저평가 구간 탈출 |
| 역매공파 | MA112/224/448 역배열 후 정배열 전환 | 장단기 추세 동반 전환 |
| 기준봉/눌림목 | MA224 돌파 후 눌림목 구간 진입 | 기준점 이탈 시 진입 |
| 데드기법 | EMA 부채꼴 전개 + 고저점 패턴 | 이동평균선 접촉 반등 |
| 다이빙 | 급등/급락 + 이동평균 근접 + 엘리엇 2파 | 기술적 반등 구간 |

**계산 지표 항목**

- 이동평균: MA5/10/20/60/112/224/448
- MA 이격도 (단기/중기/장기)
- 볼린저 밴드 (상단/하단/수렴 여부)
- CCI (Commodity Channel Index)
- 일목균형표 구름대
- 거래량 급증 감지 (N봉 내 스파이크)
- 신고가 돌파 여부
- ATR, 변동성 지표
- EMA 부채꼴 스프레드
- 주간봉 다운샘플링

### DCF 계산 `/api/valuation/calculate`

**입력 파라미터**

| 파라미터 | 설명 | 기본값 |
|---------|------|--------|
| ebitda | EBITDA (억원) | 필수 |
| netIncome | 당기순이익 (억원) | 필수 |
| netDebt | 순차입금 (억원) | 필수 |
| wacc | 가중평균자본비용 (%) | 8.5 |
| terminalGrowth | 영구 성장률 (%) | 2.0 |
| growthRate | 예측 기간 성장률 (%) | terminalGrowth x 3 또는 최소 3 |
| industry | 산업 분류 (12개) | 필수 |

**산업별 설정 (12개 산업)**

| 산업 | EV/EBITDA | PER | DCF 가중치 |
|------|-----------|-----|-----------|
| 반도체/전자부품 | 12배 | 18배 | 0.6 |
| 소프트웨어/IT 서비스 | 18배 | 30배 | 0.7 |
| 인터넷/플랫폼 | 20배 | 35배 | 0.7 |
| 제약/바이오 | 15배 | 25배 | 0.6 |
| 금융 | 8배 | 10배 | 0.4 |
| 제조/산업재 | 8배 | 12배 | 0.5 |
| 에너지/화학 | 6배 | 10배 | 0.4 |
| 통신 | 6배 | 12배 | 0.5 |
| 유통/소비재 | 10배 | 15배 | 0.5 |
| 건설/부동산 | 8배 | 10배 | 0.4 |
| 미디어/엔터테인먼트 | 12배 | 20배 | 0.6 |
| 기타 | 10배 | 15배 | 0.5 |

**출력 결과**

- DCF 가치: 기업가치, 자기자본가치, PV(FCF 합계), PV(터미널 밸류)
- 상대가치: EV/EBITDA 기반, PER 기반
- 최종 목표주가: DCF x 가중치 + 상대가치 x (1 - 가중치)
- 민감도 분석: WACC x 영구성장률 5x5 매트릭스

## 종목 점수 시스템

### 개요

한국 종목코드(6자리) 입력 시 4개 카테고리, 44개 항목을 분석하여 종합 점수를 산출합니다.

```
종합 점수 = CAN SLIM x 0.30 + 퀀트 x 0.25 + 기술 x 0.20 + 재무 x 0.25
```

**진입 단계 판정**

| 점수 | 단계 | 의미 |
|------|------|------|
| 75점 이상 | STRONG BUY | 강력 매수 |
| 60점 이상 | BUY | 매수 검토 |
| 45점 이상 | HOLD | 눌림 대기 |
| 30점 이상 | WATCH | 관망 |
| 30점 미만 | AVOID | 회피 |

|----------|------|---------|
